Understanding Manulex Morpho

In what follows, the symbols < > represent a field entry on the website. For example, <e> means that the embedded within the symbol is entered in a field of the website.

Quotation marks " : used to refer to the spelling of a word in French. For example: the word "char".

The symbols / / : used to refer to the phonetic transcription of a word or a grapheme in French, for example /SaR/ for the word "char".

The phonetic transcription relies on a simplified phonetic alphabet created for the database (correspondence between the International Phonetic Alphabet and the simplified version used in to code the database can be found on the page “Description/Phonetic codes”)

General information

• Orthographic code of the word

• Phonological code of the word

• Syntactic category (NC: noun; NP: proper noun; VER: verb; ADJ: adjective; PRO: pronoun; PRE: preposition; CON: conjunction; DET: determinant))

• Frequency of use (according to Manulex; Lété et al., 2004) provides textual frequency expressed as the number of occurrences of a word per million (value derived from the F value in the Manulex database)

• Word segmentation into its component graphemes (the symbol '.' separates two consecutive graphemes)

• Phonological segmentation of a word reflecting its segmentation into graphemes

• Grapheme-Phoneme correspondences: this field allows users to identify words that contain any given correspondence(s). In this field, a Grapheme-Phoneme correspondence will be tagged using the <-> symbol to separate a grapheme to the left from the corresponding phoneme to the right: searching for the grapheme “ch” pronounced /S/ will be entered as <ch-S>. The symbol <.> will be used to delimit a succession of Grapheme-Phoneme correspondences in a word. The sequence will be enclosed in brackets using a left parenthesis <(> to tag the beginning of a word, and a right parenthesis <)> to tag the end of a word. To search for the word “char” pronounced /SaR/, the user will enter the sequence <(ch-S.a-a.r-R)> in the field. Tags can also be used to produce a list of all the words that include a specific Grapheme-Phoneme correspondence: entering <(ch-S.> in the field, will list all the words starting with the grapheme “ch” pronounced /S/. Conversely, entering <.ch-S)> in the field, will list all the words ending with the grapheme “ch” pronounced /S/.
