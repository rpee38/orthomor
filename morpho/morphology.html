Morphological codes

The numbers 3, 4, 6 and 7 tag morphemic units in the database (remaining numbers are “machine code” tagging phonemes, see “Phonetic codes”). The complete list of morphemic units tagged in the database is presented below:

Nominal inflections for gender and number are tagged with the number 3. For example, the grapheme “s” marking plural (nominal inflection for number) in the word “photos” is coded <3s>. Its phonetic code is <3#>, as this grapheme is silent (see 'phonetic codes' section). However, the grapheme “s” in the word “parfois” is not an inflection at all, therefore it is coded <s> and its phonetic code is simply <#>.

Verbal inflections (Monophonemic) are tagged with the number 4. For example, the grapheme “ai” marks the verbal inflection in “aiderai”. It is coded <4ai>, and its corresponding phonetic code is <4E>. In the word «vrai», “ai” is not an inflection: this grapheme is coded <ai> and the corresponding phoneme <E>. Note that the coding of verbal morphology, takes into account monomorphemic G-Ph and Ph-G associations only. For example :
•It will include '4er' (chant-er), but not '4ir' ni '4oir (fin-ir, sav-oir);
•It will include '4ai' (je chant-ai) but not '4erai' (je chant-erai);
•It will include '4ez' (vous chant-ez) but not '4iez' ou '4issez' (vous chant-iez, vous fin-issez).

Silent word-ending that are no longer silent when the word is inflected or derivated: They are tagged with the number 6. For example, the final “s” in “gris” is coded <6s> and its corresponding phonetic code is <6#>. The same goes for the final “d” in «grand», with the corresponding grapheme and phoneme respectively coded <6d> and <6#>. The final grapheme is a support both for inflection (“gris” vs. “grise”/“grand” vs “grande”) and derivation (“gris” vs. “grisant”/“grand” vs. “grandeur”).

Adverbs ending in «(m)ent» are tagged by the number 7. Manners adverbials or adverbs ending in “(m)ent” mark the manner or the way something is accomplished in the sentence. For example, the grapheme “ent” in the word “simplement” is coded “7ent” and its corresponding phonetic code is <7@>.
